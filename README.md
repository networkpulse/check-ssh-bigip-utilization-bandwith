# check-ssh-bigip-utilization-bandwith
Script PERL check-ssh-bigip-utilization-bandwith qui permet de rechercher la chaine de caractères 01010045:5: Bandwidth utilization is 1070 Mbps, exceeded 75% of Licensed 1000 Mbps dans le fichier de log /var/log/ltm pour détecter un dépassement de bande passante.
